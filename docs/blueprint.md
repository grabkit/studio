# **App Name**: Blur Identity

## Core Features:

- Splash Screen: Display a splash screen with the text 'Blur' for 2 seconds on app startup.
- Get Started Screen: Navigate to a 'Get Started' screen with a button that navigates to the authentication screen.
- User Authentication: Implement user authentication with name, email, and password, including login and forgot password functionalities, using Firebase.
- Home Screen Navigation: Navigate to a home screen after successful authentication.
- Top Bar with Blur Effect: Implement a top bar with the text 'Blur' centered, using a background blur effect.
- Bottom Navigation Bar: Implement a bottom navigation bar with 'Home,' 'Post,' and 'Account' tabs.
- Account Tab: Display user details (Anonymous ID, name, email, and a profile icon) in the account tab.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to suggest a focus on identity and security.
- Background color: Very light indigo (#F0F2FA) for a clean, light theme.
- Accent color: Slightly blue violet (#7986CB) for highlighting interactive elements, such as buttons, maintaining a cool-toned theme.
- Font pairing: 'Poppins' (sans-serif) for headings, 'PT Sans' (sans-serif) for body text.
- Use material icons for a consistent and modern look.
- Ensure the application is fully mobile responsive.
- Use subtle transitions between screens and on button presses to enhance user experience.
